# 英文並び替えパズル

日本語の文を見て、バラバラになった英単語を並べ替えて英文を作る学習アプリです。

## 機能

- 毎日5問の英文並び替え問題
- 中学英語レベル（be動詞・一般動詞・疑問文・比較・不定詞など）全71問収録
- 🔥 連続クリア日数（ストリーク）表示
- 📅 過去30日間のカレンダー記録
- 進捗はブラウザに自動保存（ログイン不要）

## ローカルで動かす

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## ビルド

```bash
npm run build
```

`dist` フォルダが出力されます。Vercelはこれを自動でやってくれるので手動不要です。
